

## Mind Os update Via reading project MOUVR

### 计划目标

通过阅读。提升心智、思维，更新底层操作系统

### 操作方式

- 早上7：30到8：00，阅读一个小时
- 看书适量笔记，看完一本书做上一本书的笔记
- 手机不开机

### 选书原则

- 与心智成长相关
- 确实感兴趣
- 从上一本到下一本，产生链接
